Think like a machine!

We have a machine that can look at a picture and tell us what it is. Most of the time, it tells us the right answers. But sometimes it gives surprising answers. We don't know if this is because the machine gets confused, or if it's because the machine noticed some object or pattern hidden in the image. We've collected some of these examples here, and we want to know whether you can guess what answer the machine gave.

During the experiment, you'll see a bunch of images that will look to you like colorful television static. But the machine looked at these images and thought there were real objects hiding in there somewhere. On each trial, you'll hear about one of the labels that the machine gave; for each label, we want to know your best guess as to which image the machine thought deserved that label.
